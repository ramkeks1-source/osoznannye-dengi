# ТЗ: Приложение «6 кувшинов» — управление деньгами

**Версия:** 1.0  
**Метод:** T. Harv Eker — «Метод шести кувшинов»  
**Стек:** Vanilla HTML + CSS + JavaScript (без фреймворков)  
**Деплой:** GitHub Pages / Vercel (бесплатно)  
**Тип:** PWA (Progressive Web App) — устанавливается на телефон

---

## 1. ЦЕЛЬ ПРОЕКТА

Бесплатное веб-приложение, которое помогает пользователю распределять доходы по 6 финансовым категориям («кувшинам») согласно методу T. Harv Eker. Пользователь вводит сумму дохода — приложение автоматически рассчитывает и визуализирует распределение. Данные сохраняются локально (localStorage).

#### 5.1 Шапка (header)
- Логотип: иконка 🫙 + текст **«6 кувшинов»** (Playfair Display, gold)
- Подзаголовок: `«Метод финансовой свободы»` (Inter 300, gray)
- Справа: кнопка-иконка настроек ⚙️

#### 5.2 Блок ввода дохода
- Большой заголовок: `«Ваш доход»` (Playfair Display 32px, white)
- Input поле:
  - Placeholder: `«Введите сумму»`
  - Тип: `number`, min=`0`
  - Стиль: крупный шрифт (48px), по центру, без border (только border-bottom золотой)
  - Суффикс валюты: выпадающий список `₽ / $ / €` (по умолчанию ₽)
- Под инпутом: строка с суммой прописью (например: `«Сто тысяч рублей»`)
- Кнопка `«Рассчитать»` (золотая, rounded-full, 200px wide)
- Переключатель периода: `День / Неделя / Месяц / Год`
  - При выборе "год" — автоматически делит на 12 для расчёта месячного

#### 5.3 Сетка кувшинов
- Grid: `2 колонки на мобильном`, `3 колонки на десктопе`
- Каждая карточка кувшина:

```
┌─────────────────────────────────┐
│  [emoji]  [NAME]  [nameEn]      │
│  ──────────────────────────     │
│  [СУММА крупно]    [XX%]        │
│  [прогресс-бар]                 │
│  [description text]             │
│  [▼ примеры — раскрывающийся]   │
└─────────────────────────────────┘
```

- Цветная полоска слева (4px) — цвет кувшина
- Прогресс-бар: анимированный fill слева направо при расчёте
- Сумма: `Inter 600, 28px, white`
- Процент: badge с цветом кувшина
- При hover: `bg-hover`, лёгкое смещение вверх (`transform: translateY(-2px)`)
- **Клик по карточке** → открывает детальный drawer (см. Экран 2)

#### 5.4 Круговая диаграмма (Chart)
- Реализация: **Canvas API** (без библиотек)
- Donut chart (кольцо): внешний радиус 120px, внутренний 70px
- В центре кольца: общая сумма дохода (белый текст)
- Легенда справа: цветной кружок + название + сумма
- Анимация: сегменты рисуются по часовой стрелке при загрузке (500ms)
- При hover на сегменте: tooltip с названием и суммой

#### 5.5 История доходов
- Заголовок: `«История»`
- Список последних 12 записей (из localStorage)
- Каждая запись: дата + сумма + валюта
- Кнопка `«Очистить историю»`
- При клике на запись — восстанавливает значения в инпут

### manifest.json
```json
{
  "name": "6 Кувшинов — Финансы",
  "short_name": "6 Кувшинов",
  "description": "Метод управления деньгами T. Harv Eker",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#0D1117",
  "theme_color": "#C9A84C",
  "orientation": "portrait",
  "icons": [
    { "src": "icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

### sw.js (Service Worker)
- Кэширует: `index.html`, `manifest.json`, Google Fonts
- Стратегия: Cache First (работает офлайн)
- При обновлении: автоматически обновляет кэш

### Регистрация SW в index.html
```javascript
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js');
}
```

Реализовать:

**manifest.json** — согласно разделу 6

**sw.js:**
```javascript
const CACHE_NAME = '6jars-v1';
const ASSETS = [
  '/',
  '/index.html',
  '/manifest.json',
  'https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Inter:wght@300;400;500;600&display=swap'
];

self.addEventListener('install', e => {
  e.waitUntil(caches.open(CACHE_NAME).then(c => c.addAll(ASSETS)));
});

self.addEventListener('fetch', e => {
  e.respondWith(
    caches.match(e.request).then(r => r || fetch(e.request))
  );
});
```

**Иконки:** Создать PNG 192×192 и 512×512 — тёмный фон `#0D1117`, золотое 🫙 по центру, надпись «6» белым.

**Банер «Установить приложение»:**
- Перехватить `beforeinstallprompt`
- Показать кастомный баннер снизу: «📲 Установить как приложение» + кнопка «Установить»
- Скрыть если уже установлено

**Результат:** Приложение устанавливается на телефон как нативное.
